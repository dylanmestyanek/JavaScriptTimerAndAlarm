Voice Recognition Timer

Description: This is a timer application built with Vanilla JavaScript. While it offers preset and custom timers for convenient use, the joy of this app is using your voice to start the timer instead!

How it works: Upon clicking the 'Voice Control' button, you will be prompted with the browser asking permission to access your microphone. Once accepted, the timer will expect a voice command following this specific format:


"Set a timer for ___ minutes."

or


"Set a timer for ___ minutes and ___ seconds."

Note: The voice command has a set timer and once approved, the browser will have an 8 second window before it cancels the voice activation request. If the request times out, re-click the 'Voice Control' option, and repeat your command.

The Voice Control option will only work on the Google Chrome browser. If viewed on a mobile device, the application must be used on Android and Google Chrome.
