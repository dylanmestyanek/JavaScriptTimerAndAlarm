Voice Recognition Timer

Description: This is a timer application built with Vanilla JavaScript. While it offers preset and custom timer for convenient use, the joy of this app is using your voice to start the timer instead!

How it works: Upon clicking the 'Voice Control' button, you will be prompted with the browser asking permission to access your microphone. The timer expects a certain voice command format which should be structured as follows:


"Set a timer for ___ minutes."

or


"Set a timer for ___ minutes and ___ seconds."

Note: The voice command has a set timer and once approved, the browser will have an 8 second window before it cancels the voice activation request. If the request times out, re-click the 'Voice Control' option, and repeat your command!
